i’m using emeditor 9.17 64bit version in win7 x64 and also using total commander which is a 32bit program as file manager. i found that the explorer context menu “emeditor” is missing in total commander(but it exists in windows explorer), it is seems that this is because emeditor x64 lacks 32 bit content menu support. take winrar x64 for example, it have both 32bit and 64bit handler in HKEY\_CLASSES\_ROOT\*shellexContentMenuHandlers, while emeditor x64 only has 64bit version

Hello adamhj,  
 Thank you for using EmEditor Professional.  
 To enable both 32-bit and 64-bit context handler, I need not only add the registry settings, but also add emedshl.dll in both editions. You can install both 32-bit and 64-bit editions of EmEditor to enable context menus in both editions.
